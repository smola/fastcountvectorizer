Welcome to fastcountvectorizer
==============================

fastcountvectorizer is a faster alternative to `scikit-learn`_ `CountVectorizer`_.

.. _scikit-learn: https://scikit-learn.org/
.. _CountVectorizer: https://scikit-learn.org/stable/modules/generated/sklearn.feature_extraction.text.CountVectorizer.html

Unless noted otherwise, usage is the same as the scikit-learn's CountVectorizer.
We provide two APIs: :py:class:`fastcountvectorizer.FastCountVectorizer`, focusing on performance
and :py:class:`fastcountvectorizer.CountVectorizer`, providing higher compatibility with the
scikit-learn's CountVectorizer.
